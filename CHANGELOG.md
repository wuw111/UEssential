# UEssential v1.2.1 更新日志

**发布日期：** 2026/04/25
**当前版本：** v1.2.1
**开发版本：** v0.9.3
**适配加载器：** LeviLamina / LLBDS

---

## ✨ 新增功能

### 1. 开放跨插件联动 API (Cross-Plugin API)
为第三方开发者提供了强大的底层接口，打破插件孤岛，实现无缝对接。
- **玩家检索接口 (`queryPlayers`)：** 允许第三方插件直接调用 UEssential 的 KVDB 高可用数据库，通过模糊名称片段快速检索目标玩家的 XUID、名称及累积在线时长。
- **全局经济托管接口 (`transferMoneyApi`)：** 提供基于 API 的统一资金变动入口。第三方插件（如商店、等级系统）只需传入目标与金额，系统将自动识别在线/离线状态，并智能选择底层直扣或存入离线托管队列，完美解决第三方插件操作离线玩家资金丢失的问题。

### 2. 语言文件自动释放机制
大幅优化了国际化（I18n）系统的首次部署体验。
- **自动生成：** 首次启动或新增语言时，系统会自动将内存中的默认翻译数据导出并生成 `lang/zh_CN.json` 等物理文件，彻底解决以往版本中“找不到语言文件进行汉化/修改”的问题。

---

## 🆙 重要优化

### 1. 国际化 (I18n) 深度补全
进一步扫清了插件内的硬编码中文字符，提升插件的国际化水平。
- **全域覆盖：** 将自定义命令映射后台、离线玩家操作提示、转账队列自动结算通知，以及 API 接口触发的系统资金变动提示等，全部纳入语言配置表。
- **灵活定制：** 腐竹现在可以在语言文件中自由修改“系统通过 API 扣除了您的部分资金”等提示文本与颜色代码。

---

## 🔧 修复与常规优化
- **修复：** 修复了因 `i18n.load` 机制导致首次建服时 `lang` 文件夹内容为空，无法进行自定义修改的问题。
- **优化：** 针对 API 的模糊匹配机制进行了底层优化，确保第三方调用时能够完整获取所有重名/曾用名高度相似的玩家对象数组，保障资金操作对象的准确性。

---

## ⚠️ 提醒
- **开发者注意：** 在调用新的 `transferMoneyApi` 时，请务必优先传递 `XUID`，系统将自动向下兼容处理离线玩家的业务逻辑，无需您再编写额外的计分板或 LLMoney 适配代码。
- **语言文件更新：** 更新至此版本后，新的语言键值会自动合并到您现有的语言文件中，不会覆盖您之前已做出的翻译修改。

# UEssential v1.2.0 更新日志 (Changelog)

**发布日期：** 2026/04/18
**当前版本：** v1.2.0
**开发版本：** v0.9.2
**适配环境：** LeviLamina / LLSE 生存服务器

---

## ✨ 新增功能

### 1. KVDB 玩家数据库全面重构
底层存储引擎大换血，大幅提升海量玩家数据的读写性能。
- **平滑迁移：** 首次启动自动将旧版 `playerdatabase.json` 转换为 KVDB 格式，并生成时间戳备份。
- **自动备份：** 新增数据库自动/手动备份功能（`/playerdatabase export`），默认每日自动备份一次。
- **详细记录：** 数据库现在会精准记录玩家的“最后一次上线时间”。

### 2. 离线玩家检索与管理
打破只能管理在线玩家的限制，新增强大的模糊搜索功能。
- **模糊检索：** 通过 `/playerdatabase query [名称]` 可检索数据库中所有玩家的当前名或曾用名（限制最少3个字符防误触）。
- **离线面板：** 支持在菜单中直接对检索出的离线玩家执行“加入黑名单”与“资金调整”操作。

### 3. 离线转账与留言系统
经济系统现已支持跨越在线状态的资金流转。
- **离线转账队列：** 玩家与管理员可通过菜单搜索并向离线玩家转账，资金与消息会被存入独立的离线队列。
- **上线自动结算：** 离线玩家再次进入服务器时，系统会自动为其结算队列中的加款/扣款，并弹出历史留言通知。
- **转账留言：** 转账现支持最多 50 字的附加备注，且受到底层违禁词库的严格审查。

---

## 🛠️ 重要调整与优化

### 1. 经济接口强制覆写机制
重构了底层的经济 API 逻辑，分离了普通玩家的安全扣款与管理员的强制操作。
- **支持负数扣款：** 引入 `Eco.set` 机制，允许管理员在后台直接将玩家（含离线玩家）的计分板或 LLMoney 资金扣除至负数，彻底解决惩罚性扣款静默失败的问题。

---

## 🐞 修复与优化
- **修复：** 修复了此前管理员尝试对离线高资产玩家扣款时，因防透支安全拦截导致金币实际无变化的问题。
- **修复：** 修复了查询离线玩家时，管理菜单状态信息无法正确显示且拦截了部分合法操作的 Bug。
- **优化：** 为离线玩家查询接口加入了最小字符长度校验（`queryMinLength`），避免单字符全局检索造成的服务器瞬时卡顿。

---

## ⚠️ 提醒
- **数据迁移：** 更新至此版本后，初次启动时后台会执行数据转化，请勿在转化过程中强制关闭服务器。旧版数据会自动备份在 `plugins/UEssential` 目录下。
- **配置变动：** `config.json` 将自动合并新增的数据库备份与搜索长度限制配置节点，无需手动干预。
